# Herkulio Telegram Bot — User Guide

## What It Does

Herkulio Bot investigates people, companies, and watch dealers for you.

**In 30 seconds, it checks:**
- Corporate registry records
- Sanctions lists (OFAC, UN, EU)
- Court records & litigation
- Social media presence
- Domain & website history
- Phone/email validation
- Watch marketplace presence
- Financial red flags
- 60+ data sources total

## How to Use (Super Simple)

### Option 1: Just Type a Name
```
John Smith
```
→ Starts automatic investigation

### Option 2: Use Quick Commands

| Command | What it does | Example |
|---------|-------------|---------|
| `/v Name` | **Deep vet** (thorough check) | `/v RolexDealer Miami` |
| `/d Reference` | **Daytona price check** | `/d 116500LN` |
| `/q` | **Check your quota** | `/q` |
| `/h` | **Show help** | `/h` |

## Tips

1. **Be specific** — "John Smith Miami" better than just "John Smith"
2. **Include company type** — "LLC", "Inc", "Ltd" helps accuracy
3. **Add state/country** — For international dealers
4. **Use `/v` for big deals** — Deep search for transactions >$10K
